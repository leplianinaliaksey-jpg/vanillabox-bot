# 🎫 VanillaBox Support Bot

Telegram-бот с системой тикетов через топики группы (аналог Discord).

---

## 📁 Файлы
- `bot.py` — основной код бота
- `requirements.txt` — зависимости
- `tickets.json` — база тикетов (создаётся автоматически)

---

## 🚀 Запуск локально (Windows/Linux)

### 1. Установи Python 3.11+
https://python.org/downloads

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Запусти бота
```bash
python bot.py
```

---

## ☁️ Деплой на Railway.app (бесплатно)

1. Зарегистрируйся на https://railway.app
2. Нажми **New Project → Deploy from GitHub**
   - Загрузи файлы в репозиторий GitHub (или используй Railway CLI)
3. В настройках проекта добавь переменные окружения:
   - `BOT_TOKEN` = твой токен
   - `SUPPORT_GROUP_ID` = -1003954019230
4. Добавь файл `Procfile` в папку:
   ```
   worker: python bot.py
   ```
5. Railway автоматически запустит бота

## 🎮 Команды для игроков
| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/ticket` | Создать новый тикет |
| `/mystatus` | Посмотреть статус своих тикетов |
| `/cancel` | Отменить создание тикета |

## 🛡️ Команды для администраторов (в топике группы)
| Команда | Описание |
|---------|----------|
| Просто написать текст в топик | Ответ автоматически доставляется игроку |
| `/close <номер>` | Закрыть тикет и уведомить игрока |

---

## 💡 Как это работает

1. Игрок пишет боту `/ticket`
2. Выбирает категорию (баг, жалоба, апелляция и т.д.)
3. Вводит ник и описание проблемы
4. Бот создаёт **топик** в группе VanillaBox Support вида `🐛 #0001 | НикИгрока`
5. Любой администратор пишет в топик → игрок получает ответ в личке
6. Закрытие: `/close 0001` → топик архивируется, игрок получает уведомление
